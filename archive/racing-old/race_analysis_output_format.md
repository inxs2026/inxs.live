# Race Analysis Output Format
## Standard Format for Race Card Picks

**Updated:** February 11, 2026  
**Requirements:** Carlo's specifications for clear, consistent analysis

---

## REQUIRED ELEMENTS

### For Each Race:

**RACE X - [Race Name/Conditions]**  
**[Distance] | [Surface] | [Post Time]**

#### TOP 3 PICKS (REQUIRED)
```
1. Horse Name (odds) - Avg Beyer (last 3): XX.X
   Brief reasoning (1-2 lines)

2. Horse Name (odds) - Avg Beyer (last 3): XX.X
   Brief reasoning (1-2 lines)

3. Horse Name (odds) - Avg Beyer (last 3): XX.X
   Brief reasoning (1-2 lines)
```

#### PROSPECT (Claiming Races Only)
```
Horse Name (odds) - Avg Beyer (last 3): XX.X

IF PROSPECT IS IN TOP 3:
"Also excellent claiming value" or similar

IF PROSPECT IS NOT IN TOP 3:
MUST EXPLAIN WHY - Valid reasons:
- Outclassed today but good value for future races
- Untested at this distance but solid otherwise
- Wrong surface today, claim for preferred surface
- Consistent but facing tougher competition today
```

---

## CALCULATION: Average Beyer (Last 3)

**Formula:**
```
(Beyer Race 1 + Beyer Race 2 + Beyer Race 3) ÷ 3 = Average
```

**Example:**
```
Last 3 races: Beyers 75, 78, 72
Average: (75 + 78 + 72) ÷ 3 = 75.0
```

**What it shows:**
- Recent form level
- Consistency (tight range = reliable, wide range = volatile)
- Trend (improving vs declining)

## KEY RULES

### ✅ DO:
- Always include average Beyer (last 3) for top picks and prospects
- Explain why a prospect is NOT in top 3 (if applicable)
- Show the logic: claim = competitive, or explain the exception
- Keep reasoning brief (1-2 lines per pick)

### ❌ DON'T:
- List individual Beyers (just the average)
- Recommend claims you don't think can win without explanation
- Contradict yourself (good to claim but not good to pick)
- Forget the avg Beyer for prospects

---

## CLAIMING RACE PROSPECT LOGIC

**If prospect IS in top 3:**
- Clear recommendation
- Horse is competitive AND claimable
- Best scenario

**If prospect is NOT in top 3:**
- Must explain discrepancy
- Valid reasons ONLY:
  - Outclassed today but value for future
  - Distance/surface question
  - Consistency value vs today's competition
- Keeps analysis honest and accountable
