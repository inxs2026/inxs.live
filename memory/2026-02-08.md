## Morning (06:53)
### Racing Evaluation - New Metrics Added

Carlo expanded the prospect evaluation checklist with financial and training metrics:

**Financial Metric:**
- **Earnings per start** (total earnings ÷ starts) — measures efficiency/competitiveness
- Better than just total earnings (which favors horses with more races)
- Shows if the horse is actually competitive or just showing up

**Training Metrics:**
- **Workout times** — critical for evaluating ALL horses
  - For 2YOs just starting: basically the ONLY data available before first race
  - For experienced horses: shows current form and fitness
- **Workout spacing** (time between workouts) — indicates:
  - Training consistency
  - Recovery ability
  - Soundness (can they handle regular work?)
  - Trainer's confidence in the horse's condition

These metrics help evaluate horses with limited race history and verify current condition of experienced runners.

## Morning (07:06-07:13)
### Breeding & Surface Preference

Carlo added breeding/pedigree analysis to evaluation criteria:

**Why Breeding Matters:**
- Bloodlines give hints about preferred racing surface (dirt vs turf)
- Can predict performance before the horse proves it on track
- Helps match horse to right races/tracks
- Explains underperformance when on wrong surface

**Strategic Value:**
- Some lineages are dirt specialists, others turf-oriented
- Knowing surface preference helps spot value opportunities
- Important for picking horses suited to your available tracks

**CRITICAL INSIGHT (07:13):**
- **Wrong surface is NOT a red flag — it's an OPPORTUNITY!**
- If horse meets other criteria but struggles on current surface...
- Check breeding: might be racing on wrong surface
- **Value play:** Buy cheap → switch surface → improve performance → profit
- One of the main goals: improve performance by putting horses in right situations
- Surface mismatch can make a good horse look mediocre (= bargain for you)

## Morning (08:13)
### CRITICAL Distinction: Betting vs Claiming

Carlo emphasized key strategic difference between two uses of evaluation:

**Two different games:**
1. **BETTING** (short-term) — Care only if horse wins TODAY
2. **CLAIMING** (long-term) — Care if horse races 10+ times/year for years

**Key insight:**
- Horse with red flags CAN win a race today
- Same horse might break down immediately after
- Red flags = avoid for CLAIMING, but might be BETTING value
- "One good race left" horses = bet them, don't claim them

**Examples:**
- Voided claim + sharp works + class drop = BET ✅ / CLAIM ❌
- Missing season + bullet return work = BET maybe / CLAIM ❌
- Sound record + consistent = BET maybe / CLAIM ✅
