## TOP-3-PICKS-CRITERIA.md Review

**The methodology I follow:**
1. Calculate Beyer averages (last 3 races) - ALWAYS
2. Recent form analysis (last 3-5 races: finishes, competitiveness, trajectory)
3. Class analysis (dropping/proven/rising)
4. Trainer/jockey angles (stats from DRF)
5. Pace scenario (lone speed, closers, tactical position)
6. Surface/distance fit (breeding, past performance)

**Ranking logic:**
- #1 pick: Most positive factors (best Beyers + form + class + angles)
- #2 pick: Strong but missing 1-2 elements
- #3 pick: Viable with upside or more question marks
