## Purpose

Reference material for understanding Daily Racing Form past performances. Use when:
- Learning to read DRF PPs
- Extracting data for race analysis
- Understanding symbols and formatting
- Training on PP interpretation

## Key Takeaways

**Most Important Elements:**
1. **Running comments** (#14) - Trip notes explaining what happened
2. **Beyer Speed Figures** (#9) - Raw speed comparison
3. **Class/Conditions** (#8) - What level horse competed at
4. **Surface/Distance records** (#5) - Success at today's conditions
5. **Trainer form** (#18) - Win % in relevant categories
6. **Workouts** (#15) - Recent preparation pattern
7. **Key Race indicators** (#16) - Did horses from past races win next out?

**Critical symbols:**
- **L** = Lasix
- **÷** = First-time Lasix
- **Single line between races** = 45+ day layoff
- **Double line** = Year+ layoff
- **Italics** = Horse won next start (strong race)
- **Bold** = Horse entered today (form cycle check)
- **(v)** = Voided claim (soundness red flag)
